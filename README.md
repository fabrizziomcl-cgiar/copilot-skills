# Copilot Skills — CIP IT Support

Repositorio de **skills** reutilizables para agentes conversacionales (Copilot Studio / Teams) del equipo de IT del CIP-CGIAR.

## Estructura

```
copilot-skills/
├── skills/
│   ├── password-triage/   # Guía de triage para contraseñas y acceso
│   └── ticket-triage/     # Guía de triage para tickets de soporte
├── vercel_docs/            # Documentación de referencia
├── SKILL_example.md        # Plantilla de ejemplo para crear nuevas skills
└── README.md
```

## ¿Qué es una Skill?

Cada skill es un archivo `SKILLS.md` con instrucciones estructuradas que un agente conversacional puede seguir para atender un tipo específico de solicitud de soporte IT.

## Uso

1. Navega a `skills/<nombre-de-la-skill>/SKILLS.md`.
2. Copia o referencia el contenido en tu agente conversacional.
3. Personaliza según las necesidades de tu equipo.
